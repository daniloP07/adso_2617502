INICIANDO CON GIT

GIT es un control de versiones que permite a los equipo de desarolladores trabajar de manera remota en el mismo proyecto permite hacer contribuciones de codigo de manera local y luego replicarlos a un repositorio remoto

PRINCIPALES COMANDOS CON GIT

  • git init funciona para inicializar un proyecto de GIT.
  • git config --global user.email "email": para identificar y configurar globalmente el imail con el repositorio local.
  • git config --global user.name "name": para identificar y configurar globalmente el nombre de usuario con el repositorio local.
  • git config user.email: verificar el imail agregado
  • git config user.name: verificar nombre de usuario
  • git add funciona para agregar todos los archivos modificados y nuevos en el repositorio local de GIT.
